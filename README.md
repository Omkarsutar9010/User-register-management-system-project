# User-register-management-system-project
A full-stack User Registration Management System built using Java, Spring Boot, and React that allows users to register, login, and manage their profiles with secure backend APIs and a responsive frontend interface.

#Tech Stack
-Backend: java, spring-Boot, spring Data JPA
-Frontend: React.js
-Database: Mysql
-Tools: Git, GitHub

## Features
- User Registration
- User Login Authentication
- Form Validation
- Secure Password Storage
- REST API Integration

## Project Structure
my-fullstack-project/
├── fullstack-backend/
└── fullstack-frontend/

## How to Run

### Backend
- Open in IntelliJ
- Configure database in application.properties
- Run Spring Boot application

### Frontend
cd frontend
npm install
npm start

This project is created to demonstrate full-stack development skills.
